Git

Створити нову гілку і переходимо на неї

git checkout -b назва-гілки

З'єднати гілки в одну (merge)

  1. Записати зміни в локальний репозиторій гілки dev

    Додати файли до індексуgit add назва файлу

    Записати зміни в локальний репозиторій гілки devgit commit -m"коментар"

    Записати локальний репозиторій гілки dev на серверgit push -u origin dev

  2. Переходимо на гілку master куда ми хочемо додати код git checkout master
  3. Об'єднуємо гілку master із гілкою dev git merge dev
  4. Записати локальний репозиторій гілки master на серверgit push -u origin master
