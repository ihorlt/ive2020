Створити нову гілку і переходимо на неї

git checkout -b назва-гілки

З'єднати гілки в одну (merge)

  1. Записати зміни в локальний репозиторій git checkout master
  2. Переходимо на гілку master куда ми хочемо додати код git checkout master
  3. Об'єднуємо гілку master із гілкою dev git merge dev
